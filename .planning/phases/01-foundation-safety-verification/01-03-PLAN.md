## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Registry data to HTML strings | Third-party text and URLs are interpolated into `innerHTML` renderers. |
| DOM attributes to app state | `data-view` values cross from static HTML into TypeScript state. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-03-01 | Tampering | `focusView.ts` and `componentView.ts` links | mitigate | Parse URLs with `new URL()`, allow HTTPS only, render invalid links as unavailable text. |
| T-01-03-02 | Information Disclosure | `shell.ts` render error fallback | mitigate | Log exception details only to console and use neutral UI-SPEC copy in visible fallback. |
| T-01-03-03 | Elevation of Privilege | String-rendered HTML sinks | mitigate | Escape `&`, `<`, `>`, `"`, and `'` through a single shared helper covered by tests. |
| T-01-03-04 | Tampering | Tab `data-view` attributes | mitigate | Add an `isView` guard and ignore invalid view values. |
</threat_model>
